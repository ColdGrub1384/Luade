# Luade

```
Luade is a Lua IDE for iPhone and iPad. You can run code directly on your device and offline.

Features:

    - Lua with all default libraries like "io"
    - Full REPL
    - Use your own modules
```

## Building

`$ ./setup.sh` and then open Luade.xcworkspace.
